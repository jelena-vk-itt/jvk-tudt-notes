  2. Long theory questions [1 x 20 marks on exam paper]

    Long theory questions [1 x 20 marks on exam paper]

    In the exam, the answer needs to be a critical discussion, with reference to various listed aspects of the topic, e.g. the relevant context, associated technologies, deployment and areas of application.

    1. What is the difference between full-stack development and client-side development? Critically discuss with reference to technologies, applications, component and deployment architectures. Include a diagram.
    2. What are client-side rendering (CSR), static-site generation (SSG) and server-side rendering (SSR) in the context of client-side Web development and how do they compare to each other? Discuss with reference to the relevant context, development, deployment, problem types to which each rendering approach is most applicable and include examples (article).
    3. What is client-side web development? Define and discuss with reference to development, deployment, technologies, suitable applications and anything else you know about it.
    4. What are Web APIs? Define and discuss with reference to where they are found and how they are used in Web development. Give examples of five Web APIs you have used in Web development, explaining their purpose and providing some examples of use with some code (or pseudocode).

  3. Short theory questions [6 x 5 marks on exam paper]

    Short theory questions [6 x 5 marks on exam paper]

    In the exam, these questions need to be answered briefly (two or three sentences at most), but in a way that demonstrates your understanding of the relevant concepts.

    1. What is the viewport and what is one purpose of setting its properties with a HTML meta tag?
    2. What is the difference between a framework and a library?
    3. What is cross-origin resource sharing (CORS)? Explain how it works.
    4. What is the JavaScript engine?
    5. How do asynchronous requests with conventional callbacks and promises differ in terms of JavaScript engine queues?
    6. What is the difference between the Internet and the World Wide Web?

  4. Long applied questions [1 x 20 marks on exam paper]

    Long applied questions [1 x 20 marks on exam paper]

    In the exam, this question will describe an application in terms of a set of requirements and ask you to write a technology plan for its realisation. The required focus with respect to development stages (design, implementation, deployment, maintenance etc.) will be specified in the question. You will be asked to give both generic names and examples of specific technology products. Most of these questions will also require a diagram in the answer.

    1. You are managing a project to implement a game of 30 seconds, with the following requirements:
      • the game must be implemented as a static Web application
      • it should be usable on both computer browsers and mobile devices
      • the user should be able to create their own concept bank, save it and share it with other users
      • the game uses a 30s timer
      • the game should access concept banks as they are needed
      • multiple developers with frequent updates to the application must be facilitated
      Write a technology plan for this project, from the point of view of both development and any relevant operations. This should include as much detail as possible on technologies, how they are used and how they fit together. The plan should include a diagram, use generic technology names and give examples of specific products where relevant.
    2. You have been asked to implement a children's game using client-side web technologies. The requirements are as follows:
      • the game must be implemented as a static Web application
      • it will be used by sighted children and children with slight visual impairment, often needing pictures or text to be enlarged
      • the children must be able to use a mouse to move objects around the screen
      • the game should be usable both from a PC browser and from a mobile device
      • a child playing the game should be able to save the game and resume playing at a different time (on the same device)
      • data for the game is substantial and stored separately in one or more file; it is not known in advance which portions of the data will be needed
      • the game needs to be developed quickly (so you ask a friend to help you)

  5. Short applied questions [6 x 5 marks on exam paper]

    Short applied questions [6 x 5 marks on exam paper]

    1. Which of the strings 'HTTP://ABC.COM', 'https://xyz.com', 'http://aaacom', if any, match the following regular expression? Explain. 
      
/^http[s]?:\/\/[a-z]+.com/i
    2. Which of the elements in the following HTML snippet, if any, correspond to the CSS selector p.special_element#abc
      
<p id="special_element#abc">1 Hello<p>

<p class="special_element#abc">2 World<p>

<p class="special_element" id="abc">3 How<p>

<div class="p.special_element#abc">4 Are<p>

<div class="p special_element" id="abc">5 You<p>
      Explain your answer.
    3. Explain the effect of the following media query for all ranges of relevant properties.
      
@media screen and (max-width: 600px) {
    body { background-color: red; }
}
