Creating the GitHub Repostiory for Lab Work

All lab work is to be uploaded to your private repository in GitHub Classroom CSWD-SWD-FT_2023-24-S1. To create this repository, do the following:

  1. If you do not have a GitHub account, create one on github.com. Use your mytudublin.ie email address.
  2. Click on this GitHub Classroom invitation link.
  3. If you see the following page, log into GitHub (if you do not have an account, go back to step 1).
  4. If you see the following page, click "Authorize GitHub Classroom".
  5. On the following page, pick your X-number. If your X-number does not appear on the list, pick one of the ids that starts with xNOIDYET.
  6. When the pop-up shown below appears (it might look a bit different in a different browser), check that the id number is the one you wanted to pick and click 'Ok'.
  7. On the next page click 'Accept this assignment'.
  8. Next you might see the page shown below. Wait for your assignment to be configured. If the page does not change after a minute or two, refresh it (F5).
  9. Now you should see the page shown below. The given URL is the URL of your repository for CSWD labs. Click on the assignment URL (with blue background) to open the repository.
  10. You will see the following page telling you how to initialise the repository. Click on 'README' (blue text).
  11. The following page allows you to edit the README file before saving. Type in your real name here if your GitHub id does not identify you. Click 'Commit changes'.
  12. A dialogue window pops up, allowing you to specify commit comments. You can use the default. Click 'Commit changes'.
  13. The next page shows your initialised repository. You will keep all your CSWD lab work here and I, the lecturer, can see it. You can bookmark this page for easier access in the future.
