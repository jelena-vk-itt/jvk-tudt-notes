Git repository working directory corrupted or in 'funny' state

If you are having problems with git between classes that you cannot otherwise solve, here is a 'hack' that will allow you to 'restart'.

  • Rename the existing working directory (cswd-labs-<your-github-name>) to something else (e.g. cswd-labs-<your-github-name>-old)
  • Clone again
  • Copy any uncommited work from the old directory (cswd-labs-<your-github-name>-old) to the new one (cswd-labs-<your-github-name>)
  • Perform the add, commit and push from the new directory (cswd-labs-<your-github-name>)
