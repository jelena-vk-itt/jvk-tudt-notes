Python built-in container types

There are four types of built-in container types in Python:

  • lists
  • tuples
  • sets
  • dictionaries

So far in the module we have used the first two. In this session we will look at sets and dictionaries.

  • The Python set container contains unique elements (like in maths). It can be used, among other things, for testing membership and identifying unique values in lists.

    Open the Python command line interface and, with reference to the documentation, complete the following tasks.

    • Create a set with each of the three set creation methods. One pair of these sets should have some overlapping elements. The third should be a subset of one of the other sets.
    • Get the length of a set and test some elements for membership in a set (try with an element that you know is in the set and with an element that is not).
    • Perform at least three set operations and for each call the corresponding methods with the same arguments.

    EXAMPLE SOLUTION

    Create a new Python script and in it save the following list into a variable:

    [1, 1, 2, 23, 1, 3, 6, 7, 7, 8, 23, 1, 1, 9, 2, 22, 34]

    Write some code that uses a set container to count the number of unique values in the list.

    EXAMPLE SOLUTION

  • The Python dict container is a mapping object, which maps unique keys to arbitrary objects.

    Create a Python script and, using the first method for dictionary creation given in the documentation, create a dictionary object containing the following data:

    {
"apple": "a fruit, can be green, red or yellow",
"baby": "a very young person", 
"car":	"a transport vehicle",
"deer":	"an animal with antlers",
"earth": "a planet, the one we live on"
}

    In the same script, print the dictionary, the perform the following operations:

    1. create a list of the keys in the dictionary and print it
    2. create a list of the values in the dictionary and print it
    3. print the number of entries in the dictionary
    4. lookup 'car' and 'earth' and for each print the result
    5. check if 'baby' is in the dictionary and print the result
    6. delete 'baby' from the dictionary, then print the dictionary
    7. call the popitem method on the dictionary, print the result and the dictionary
    8. call the items method on the dictionary and iterate through the result, printing out each item
    9. call the clear method on the dictionary, then print the dictionary
