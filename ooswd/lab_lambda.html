Exercise 1: Game results

Write a Python program that does the following

  • reads user-input names of players in a game
  • reads user-input scores for the players (these are integers and a higher score means a better result)
  • finds and prints the highest score
  • finds and prints the lowest score
  • finds and prints the average score
  • sorts and prints the list of players in order of their scores

Tips:

  • The program must work for any number of players.
  • You can use the functions min(), max() and sum() to find the score statistics that are required.
  • The easiest way to sort the players is with the method sort() or function sorted() (sorted() does the same thing as sort() but does not modify the original list, returning a copy). To sort by score, you will need to use a key argument and as the conversion from name to score is a bit more complicated than the examples we covered in class, please look up the list notes (the section on sorting), which have been updated with information on lambda expressions (I know I said in the lecture that we would not be covering this but I realise sorting would be very limited without them).
