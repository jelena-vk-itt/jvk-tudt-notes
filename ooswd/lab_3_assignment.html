Some notation information

  • Angular brackets containing descriptive text are placeholders, used to describe required input or other literals. For example,
    • if an expected input string value is described as <name> then the actual value could be John or Mary
    • if a value is described as <date of birth in format dd/mm/yy> then the actual value might be 01/01/01

    The actual value should not include angular brackets.

  • If a command line window is shown to demonstrate requirements,
    • text printed by the programe is displayed in white
    • user input is displayed in red
    For example,
    Please enter a number: 42

General instructions

  • In the submitted code you must use only
    • the constructs covered in class so far
    • any additional string and list methods you might need
    • any additional constructs explicitly required by the exercise
  • Unless otherwise specified, you can assume a well behaved user, who always enters valid input.
  • Reading of user input must always be accompanied by a suitable textual prompt.
  • Printing of output must always be accompanied by a suitable textual heading or explanation.

Exercise 1: Game results

Write a Python program that does the following

  • reads user-input names of players in a game
  • reads user-input scores for the players (these are integers and a higher score means a better result)
  • finds and prints the highest score
  • finds and prints the lowest score
  • finds and prints the average score
  • sorts and prints the list of players in order of their scores

Tips:

  • The program must work for any number of players.
  • You can use the functions min(), max() and sum() to find the score statistics that are required.
  • The easiest way to sort the players is with the method sort() or function sorted() (sorted() does the same thing as sort() but does not modify the original list, returning a copy). To sort by score, you will need to use a key argument and as the conversion from name to score is a bit more complicated than the examples we covered in class, please look up the list notes (the section on sorting), which have been updated with information on lambda expressions (I know I said in the lecture that we would not be covering this but I realise sorting would be very limited without them). I will also post a short video with explanations in the recording section on Moodle.

Exercise 4: Home insulation grant

Write a Python program that does the following

  • reads a house floor area (in sqm) input by the user
  • reads a house type (detached, semi or terraced) input by the user
  • calculates and informs the user of the amount that would be granted for external insulation if:
    • the grant per sqm for the different house types is 2.5, 1.5 and 1 euro for detached, semi and terraced respectively
    • there is a cap of 150sqm on the size for which the grant is given (a house larger than 150sqm gets a grant only for 150sqm)

Exercise 5: Find instances

Write a Python program that does the following:

  • reads in two strings
  • finds all the non-overlapping instances of the second string in the first string
  • prints the index positions in the first string at which the second string is found
Example command line interaction:
Please enter a string: Bananas, oranges and mangos.
Please enter the substring to search for: an
The substring appears at positions 1, 3, 11 and 17 in the string.

Exercise 6: Password

Write a Python program that:

  • stores a string literal in a variable called password
  • uses a while loop to read in a password string typed in by the user, giving them three tries to get it right by typing in the correct password
  • if the password is correct in one of the first three iterations, prints a message telling the user that they are logged in
  • if the password is incorrect three times, prints out an error message
Example command line interaction (password is "abc"):
Please enter the password: xyz
The password you entered is not correct. Please enter the password: hello
The password you entered is not correct. Please enter the password: abc
Thanks. You are logged in now.
Another example command line interaction (password is "abc"):
Please enter the password: xyz
The password you entered is not correct. Please enter the password: hello
The password you entered is not correct. Please enter the password: 123
You have entered an incorrect password too many times. Please contact the administrator.
