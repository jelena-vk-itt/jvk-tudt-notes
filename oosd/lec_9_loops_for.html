For loops

  1. Ranges

    Ranges

    The range is an immutable sequence type that allows easy definition of integer sequences. We introduce it here as for loops in Python are often defined with the use of ranges.

  2. For loop syntax in Python

    For loop syntax in Python

    In Python, a for loop iterates over the members of any iterable object. A statement block is defined, like in the while loop, with instructions to be executed at each iteration.

    The picture shows the flowchart for a for loop, alongside the generic flow chart.

  3. Using for loops

    Using for loops

    Sequence types are iterable objects in Python. This means that we can design for loops that iterate over any of the sequence types we have covered so far.

    In terms of the types mentioned in chapter While loops, section Exiting the loop, it is the iteration processes for which the iterations are determined before execution starts (either at coding time or at run time) that are suitable for implementation using for loops.
